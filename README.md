## Setup

**DO NOT COMMIT `config.env` TO REPO**

1. cp `config.env.example` `config.env`
2. update the contents of `config.env`

3. `nvm use`
4. `npm install`
5. `npm run dev`

## Database Seeder

seed data

```
node seeder -i
```

delete data

```
node seeder -d
```
